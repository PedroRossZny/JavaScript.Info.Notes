# JavaScript Basic operators, maths

## Exercise 028

Um operando é aquilo a que os operadores são aplicados, como em `5 * 2`, os operandos são `5` e `2` e o operador nesse caso é `*`, o operador pode ser considerado *unário* quando tiver um unico operando, como o `-` para negação, um operador é binário caso tenha dois operandos, o mesmo sinal de menos também existe na sua forma binária, para subtração. Como mostrado no **ex028**.

## Exercise 029

As seguintes operações matematica são suportadas: Adição `+`, Subtração `-`, Multiplicação `*`, Divisão `/`, Restante `%` e Exponenciação `**`. O operador `%` retorna o resultado do resto da divisão de um operando por outro, o operador `**` eleva o primeiro operando à potência do outro, também podemos usa-lo para calcular raizes, elevando a números como `1/2`. Como mostrado no **ex029**.

## Exercise 030

O operador `+` além de somar números ele também pode concatenar strings, em uma operação onde temos um número por outro número mas em formato de string, tudo será convertido para uma string, em um caso mais complexo é em uma operação com mais de 2 operandos, onde é seguido a ordem da operação, quando chegar na string, tudo que foi operado até o momento será convertido para uma string, o operador `+` é o unico que suporta string dessa maneira, outros operadores funcionam apenas com números e sempre convertem seus operandos em números. Como mostrado no **ex030**.
