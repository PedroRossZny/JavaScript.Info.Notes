# JavaScript Script Inclusion

## Exercice 001

JavaScript programs can be inserted anywhere in the code using the `<script>` tag, as shown in **ex001**.

---

## Exercice 002

If the JavaScript code grows to large proportions, we can place it in a separate `.html` file using the `src` attribute. We can specify the full path to the script starting from the project's root directory, or we can provide a relative path using only the file name if it is located in the same folder as the `.html` file, as shown in **ex002**.

---

## Exercice 003

You can also attach multiple scripts to the same `.html` file. They only need to have different names, and each one must be included in its own `<script>` tag, as shown in **ex003**.

---

## Exercice 004

If the `src` attribute of a `<script>` tag is defined, the content inside that same tag will be ignored. If you want to do something like this, you must separate it into two scripts: one external and one internal, as shown in **ex004**.
